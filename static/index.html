API Routes

Home Page

HTTP Methods Path Description
GET / Home page

Users

HTTP Methods Path Description
GET /user Get all users
GET /user/:id Get user by ID
POST /user/register Register a new user
PATCH/PUT /user/:id Update user by ID
DELETE /user/:id Delete user by ID

Products/Items

HTTP Methods Path Description
GET /product Get all products
GET /product/recent Get 5 most recent products
GET /product/:id Get product by ID
POST /product/add Add a new product
PUT/PATCH /product/:id Update product by ID
DELETE /product/:id Delete product by ID

Carts

HTTP Methods Path Description
GET /carts Get all carts
GET /:id/cart Get cart from user
POST /:id/cart Add items in cart
PATCH /:id/cart/:prodID Update items in cart
DELETE /:id/cart Delete items from the cart
Delete /:id/cart/:prodID Delete a specific item from the cart.
